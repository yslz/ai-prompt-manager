# AI Prompt Manager

管理你的 AI 角色和 Prompt，一键复制到剪贴板！

## 🎯 解决的问题

- 每次都要重新输入"你是一个..."？
- 有多个常用角色，切换麻烦？
- 在不同 AI 平台间重复配置？

这个插件帮你**一次配置，到处使用**！

## ✨ 功能

- 🎭 **角色管理** - 创建多个 AI 角色
- 📋 **一键复制** - 点击即可复制到剪贴板
- 🌐 **跨平台** - 支持所有 AI 平台
- 💾 **本地存储** - 数据安全，不上传


## 🚀 安装方法

### Chrome/Edge

1. 打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `ai-prompt-manager` 文件夹
5. 完成！

## 📖 使用指南

### 1. 创建角色

```
1. 点击插件图标
2. 点击"➕ 添加新角色"
3. 填写：
   - 角色名称：代码专家
   - 图标：💻
   - Prompt 内容：你是一个资深程序员...
4. 点击"保存"
```

### 2. 使用角色

```
方法 1: 侧边栏
1. 在任何 AI 页面，右侧显示侧边栏
2. 点击角色的"📋 复制"按钮
3. 在 AI 输入框中粘贴 (Ctrl+V)

方法 2: 插件图标
1. 点击插件图标
2. 点击角色的"📋 复制"按钮
3. 在 AI 输入框中粘贴 (Ctrl+V)
```

### 3. 编辑/删除角色

```
1. 点击插件图标
2. 点击角色的"✏️ 编辑"或"🗑️ 删除"
```

## 📝 示例角色

### 代码专家
```
你是一个资深的软件工程师，擅长：
- 代码审查和优化
- 架构设计
- 调试和解决问题
- 最佳实践建议

请用专业的态度回答我的编程问题。
```

### 写作助手
```
你是一个专业的编辑和写作助手，擅长：
- 文章润色和修改
- 语法和拼写检查
- 文风调整
- 结构优化

请帮我改进文章的质量。
```

### 翻译专家
```
你是一个专业的翻译，擅长：
- 中英文互译
- 保持原文风格
- 专业术语准确
- 流畅自然的表达

请帮我翻译以下内容，保持准确和流畅。
```

## 🔧 高级功能

### 快捷键（可选）

在 `chrome://extensions/shortcuts` 中设置快捷键，快速复制最常用的角色。

### 导入/导出

角色数据存储在本地，可以手动备份 `chrome.storage.local` 中的数据。

## 📄 隐私

- ✅ 所有数据**本地存储**
- ✅ **不上传**到任何服务器
- ✅ **不收集**任何个人信息
